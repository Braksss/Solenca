# Solenca

Plateforme digitale pour simplifier l’immobilier en Espagne pour les Français.

## Structure

- \pps/frontend\ → Interface React
- \pps/cms-strapi\ → CMS Strapi pour la gestion de contenu

## Démarrage

\\\ash
cd apps/cms-strapi
pnpm install
pnpm run develop
\\\

\\\ash
cd apps/frontend
pnpm install
pnpm run dev
\\\
